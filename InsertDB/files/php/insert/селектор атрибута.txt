<div class="title">Селектор атрибута</div>
<div class="text"Селекторы атрибутов выбирают элементы на основе имени атрибута или значения атрибута:

<br><hrh>[атрибут]</hrh> — все элементы, содержащие указанный атрибут, <hrh>[alt]</hrh> — все элементы, для которых задан атрибут <hrh>alt;</hrh>
<br><hrh>селектор[атрибут]</hrh> — элементы данного типа, содержащие указанный атрибут, 
<br><hrh>img[alt]</hrh> — только картинки, для которых задан атрибут alt;
<br><hrh>селектор[атрибут="значение"]</hrh> — элементы данного типа, содержащие указанный атрибут с конкретным значением, img[title="flower"] — все картинки, название которых содержит слово flower;
<br><hrh>селектор[атрибут~="значение"]</hrh> — элементы частично содержащие данное значение, например, если для элемента задано несколько классов через пробел, 
<br><hrh>p[class~="feature"] </hrh>— абзацы, имя класса которых содержит feature;
<br><hrh>селектор[атрибут|="значение"]</hrh> — элементы, список значений атрибута которых начинается с указанного слова, p[class|="feature"] — абзацы, имя класса которых feature или начинается на feature;
<br><hrh>селектор[атрибут^="значение"]</hrh> — элементы, значение атрибута которых начинается с указанного значения, a[href^="http://"] — все ссылки, начинающиеся на http://;
<br><hrh>селектор[атрибут$="значение"]</hrh> — элементы, значение атрибута которых заканчивается указанным значением, img[src$=".png"] — все картинки в формате png;
