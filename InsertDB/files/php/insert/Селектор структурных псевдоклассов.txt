<div class="title">Селектор структурных псевдоклассов</div>
<div class="text">Структурные псевдоклассы отбирают дочерние элементы в соответствии с параметром, указанным в круглых скобках:
<br>
<br><hrh>a:link</hrh> — не посещенная ссылка;
a:visited — посещенная ссылка;

<br>:nth-child(odd) — нечётные дочерние элементы;

<br>:nth-child(even) — чётные дочерние элементы;

<br>:nth-child(3n) — каждый третий элемент среди дочерних;

<br>:nth-child(3n+2) — выбирает каждый третий элемент, начиная со второго дочернего элемента (+2);

<br>:nth-child(n+2) — выбирает все элементы, начиная со второго;

<br>:nth-child(3) — выбирает третий дочерний элемент;

<br>:nth-last-child() — в списке дочерних элементов выбирает элемент с указанным местоположением, аналогично с :nth-child(), но начиная с последнего, в обратную сторону;

<br>:first-child — позволяет оформить только самый первый дочерний элемент тега;

<br>:last-child — позволяет форматировать последний дочерний элемент тега;

<br>:only-child — выбирает элемент, являющийся единственным дочерним элементом;

<br>:empty — выбирает элементы, у которых нет дочерних элементов;

<br>:root — выбирает элемент, являющийся корневым в документе — элемент html.
