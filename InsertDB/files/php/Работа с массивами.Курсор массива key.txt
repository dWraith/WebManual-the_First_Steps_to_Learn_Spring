<div class="title">key</div>
                <h1>Описание:</h1>
                <div class="text">
      Функция возвращает индекс текущего элемента массива. 
                </div>
                 <h1>Синтаксис:</h1>
                <div class="syntax">
   	mixed key(array arr)
                </div>
		<div class="text">
Функция key() возвращает индекс текущего элемента массива. 
<br><h1>Пример использования функции key(): </h1> 
                </div>
		<div class="syntax">
   	<?php
<br>$array = array( 
 <br>  "fruit1" => "apple",

<br>"fruit5" => "apple");
<br>// этот цикл пройдется по всему массиву
