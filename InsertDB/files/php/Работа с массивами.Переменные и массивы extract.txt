<div class="title">extract</div>
                <h1>Описание:</h1>
                <div class="text">
  Импорт элементов массива в переменные
                </div>
                 <h1>Синтаксис:</h1>
                <div class="syntax">
 int extract (array var_array [, int extract_type [, string prefix]])
                </div>
		<div class="text">
	Функция extract() используется для импортирования значений массива в строку. Эта функция берет ассоциативный массив var_array, использую ключ элемента как имя переменной, а значение элемента как значение полученной переменной. Для создания переменных из пары ключ => значение можно также использовать необязательные параметры extract_type и prefix. 
Замечание: начиная с PHP версии 4.0.5 эта функция возвращала количество извлеченных переменных. 
Функция extract() проверяет, действительно ли каждый ключ массива может быть названием переменной. Она также проверяет на наличие одинаковых названий получаемых переменных. В случае, если обнаруживаются недействительные или числовые ключи, поведение функции предопределяет параметр extract_type, который может принимать следующие значения: 
<br>EXTR_OVERWRITE 
<br>Если переменная с таким именем уже существует, то ее значение будет заменено (значение по умолчанию). </p>
EXTR_SKIP 
<p>Если переменная с таким именем уже существует, то ее значение не будет заменено. 
<br>EXTR_PREFIX_SAME 
<p>Если переменная с таким именем уже существует, то к ее имени добавится префикс, определенный в параметре prefix. 
<br>EXTR_PREFIX_ALL 
<p>Ко всем образуемым переменным будет добавлен префикс prefix. 
<br>EXTR_PREFIX_INVALID 
<p>Только при неправильных или числовых индексах массива к имени переменной будет добавлен префикс prefix. 
<br>EXTR_IF_EXISTS 
<p>Если переменная с таким именем уже существует, то ее значение будет переписано. Иначе ничего не сделает. Это может быть полезно, если вы задали список всех переменных, а затем хотите извлечь только те переменные, которые были определены, например, из $_REQUEST. Этот флаг был добавлен в PHP 4.2.0 
<br>EXTR_PREFIX_IF_EXISTS 
<p>Создает переменные с префиксом, если такие переменные без префикса уже существовали. В любом другом случае ничего не создает. Этот флаг был добавлен в PHP 4.2.0 
<br>EXTR_REFS 
<p>Переменные создаются как ссылки. Это означает, что созданные переменные все еще ссылаются на значения соответствующих элементов массива var_array. Вы можете применять этот флаг как в одиночку, так и совместно с другими флагами. Этот флаг был добавлен в PHP 4.3.0 
                </div>
		<div class="syntax">
	<?php
<br>/* Предположим, что $var_array массив,    
<br>полученный при помощи функции wddx_deserialize */
<br>$size = "large";
<br>$var_array = array("color" => "blue",        
<br>           "size"  => ""medium",                
<br>     "shape" => "sphere");
<br>extract($var_array, EXTR_PREFIX_SAME, "wddx");
<br>echo "$color, $size, $shape, $wddx_sizen";
<br>?>
                </div>
